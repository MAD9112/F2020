# H6 - VM Screenshot

# Task

Open your VM running Ubuntu. Find the terminal app and run it. Create an enpty file and use lastName-firstName.txt as its name. Use ls to check if the file has been created.

Take a screenshot of the whole screen.

Name your screenshot \<lastName>-\<firstName>-H6.png

Upload the image to Brightspace.

# Acceptance

By looking at the picture, I have to be able to see you were in fact running Ubuntu in a VM

and

I have to be able to see you created the requested file with the correct name.
